## :information_source: Como utilizar:

```bash
# Clone esse repository
$ git clone https://github.com/martinsalmeida/clone-twitter

# Entre na pasta
$ cd clone-twitter

# Instale as depêndendias
$ yarn install

# Execute o app 
$ yarn start
```
